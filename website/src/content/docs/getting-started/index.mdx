NeoHaskell lives in the [neohaskell/NeoHaskell](https://github.com/neohaskell/NeoHaskell)
monorepo. This documentation site is intentionally small; the authoritative setup,
build, and test instructions live in the repository's `README.md`.

## Quick start

1. Install [Determinate Nix](https://determinate.systems/nix).
2. Clone the repository and enter its development shell:

   ```bash
   git clone https://github.com/neohaskell/NeoHaskell.git
   cd NeoHaskell
   nix develop
   ```

3. Run `./dev` to list the available build and test commands.

For the full workflow — Postgres, the test suite, and the contribution process — see
the [repository README](https://github.com/neohaskell/NeoHaskell#readme).
