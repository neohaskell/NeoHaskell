NeoHaskell は [neohaskell/NeoHaskell](https://github.com/neohaskell/NeoHaskell) モノレポにあります。このドキュメントサイトは意図的に小さく作られており、信頼できるセットアップ・ビルド・テストの手順はリポジトリの `README.md`（英語）にあります。

## クイックスタート

1. [Determinate Nix](https://determinate.systems/nix) をインストールします。
2. リポジトリをクローンし、`nix develop` で開発シェルに入ります。
3. `./dev` を実行して、利用可能なビルド・テストコマンドを確認します。

完全なワークフロー（Postgres、テストスイート、コントリビューションの流れ）については、[リポジトリの README](https://github.com/neohaskell/NeoHaskell#readme) を参照してください。
