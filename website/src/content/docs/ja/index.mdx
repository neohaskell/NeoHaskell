## NeoHaskell とは

NeoHaskell は、初心者と実用主義者のために設計された Haskell の方言です。イベントソーシングを言語の第一級の機能にします。イベント、エンティティ、コマンド、クエリ、サービス、インテグレーションはライブラリのパターンではなく、言語のプリミティブです。状態遷移が誤っていれば、プログラムはコンパイルされません。
