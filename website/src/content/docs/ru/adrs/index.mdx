Записи об архитектурных решениях (ADR) документируют значимые архитектурные решения, принятые при создании NeoHaskell.

Записи ADR ведутся канонически **на английском языке** и не переводятся, чтобы существовал единый источник истины. Полный список — по одной странице на каждое решение:

- **[Записи об архитектурных решениях (на английском)](/adrs/)**
