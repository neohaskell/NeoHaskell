NeoHaskell находится в монорепозитории [neohaskell/NeoHaskell](https://github.com/neohaskell/NeoHaskell). Этот сайт документации намеренно небольшой; авторитетные инструкции по установке, сборке и тестированию находятся в файле `README.md` репозитория (на английском языке).

## Быстрый старт

1. Установите [Determinate Nix](https://determinate.systems/nix).
2. Клонируйте репозиторий и войдите в его среду разработки с помощью `nix develop`.
3. Запустите `./dev`, чтобы увидеть доступные команды сборки и тестирования.

Полный рабочий процесс — Postgres, набор тестов и процесс участия — описан в [README репозитория](https://github.com/neohaskell/NeoHaskell#readme).
