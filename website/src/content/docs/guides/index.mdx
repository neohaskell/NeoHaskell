The [tutorial](/tutorial/) teaches you the language. The [concept pages](/concepts/) explain the ideas. This section is for the real problems you'll hit once you start shipping: testing strategies, deployment, integrating with the outside world, adopting event sourcing at work, handling errors, and dealing with the kinds of things tutorials skip.

Each guide is self-contained. Read the ones you need, skip the rest, come back later.
