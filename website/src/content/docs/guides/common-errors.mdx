### `Couldn't match type 'EntityState' with 'Maybe EntityState'`

{/*
  Forgetting that entity state is wrapped in Maybe (entity might not exist yet).
*/}


- **What happened**: you're treating the entity state as if it always exists, but it might be `Nothing` (entity not yet created)
- **Fix**: pattern match on `Nothing` (entity doesn't exist) and `Just state` (entity exists) separately
- **Why**: entities start as `Nothing` — the first command creates them

## I Was Trying to Test Something

### Test assertion failure: `Expected [EventA, EventB], got [EventA]`

{/*
  The test expected more events than were produced.
*/}


- **What happened**: your command handler produced fewer events than the test expected
- **Debugging steps**:
  - Check the command handler logic — is there a condition that prevents the second event?

### `Address already in use (port 8080)`

- **What happened**: another process is already using the port your app wants to bind to
- **Fix options**:
  - Kill the other process: `lsof -ti:8080 | xargs kill`
  - Change your app's port in the configuration
- Show: how to change the port in `defineConfig`

### `Event store connection failed`

- **What happened**: the app can't connect to the event store (PostgreSQL or in-memory)
- **Fix for in-memory**: this shouldn't happen — check if you accidentally configured PostgreSQL
- **Fix for PostgreSQL**: check the connection string, check that PostgreSQL is running
- Show: the configuration for each event store type

### `neo: command not found`

- **What happened**: NeoHaskell is not installed or not in your PATH
