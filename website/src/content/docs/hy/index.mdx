## Ի՞նչ է NeoHaskell-ը

NeoHaskell-ը Հասքելի բարբառ է՝ նախատեսված սկսնակների և գործնական մարդկանց համար։ Այն event sourcing-ը դարձնում է լեզվի առաջնային հատկություն. իրադարձությունները, էությունները, հրամանները, հարցումները, ծառայությունները և ինտեգրումները լեզվի պարզունակ տարրեր են, ոչ թե գրադարանային ձևանմուշներ։
