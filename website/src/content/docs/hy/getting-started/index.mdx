NeoHaskell-ը գտնվում է [neohaskell/NeoHaskell](https://github.com/neohaskell/NeoHaskell) մոնոշտեմարանում։ Այս փաստաթղթերի կայքը միտումնավոր փոքր է. տեղադրման, կառուցման և թեստավորման հիմնական հրահանգները պահվում են շտեմարանի `README.md`-ում (անգլերեն)։

## Արագ սկիզբ

1. Տեղադրեք [Determinate Nix](https://determinate.systems/nix)։
2. Կլոնավորեք շտեմարանը և մուտք գործեք նրա մշակման կեղև՝ `nix develop`։
3. Գործարկեք `./dev`՝ հասանելի հրամանները տեսնելու համար։

Ամբողջական աշխատանքային հոսքի համար տե՛ս [շտեմարանի README-ն](https://github.com/neohaskell/NeoHaskell#readme)։
