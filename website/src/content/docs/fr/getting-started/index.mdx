NeoHaskell vit dans le monorepo [neohaskell/NeoHaskell](https://github.com/neohaskell/NeoHaskell). Ce site de documentation est volontairement minimal ; les instructions de référence pour l'installation, la compilation et les tests se trouvent dans le `README.md` du dépôt (en anglais).

## Démarrage rapide

1. Installez [Determinate Nix](https://determinate.systems/nix).
2. Clonez le dépôt et entrez dans son shell de développement avec `nix develop`.
3. Lancez `./dev` pour lister les commandes de compilation et de test disponibles.

Pour le flux de travail complet — Postgres, la suite de tests et le processus de contribution — consultez le [README du dépôt](https://github.com/neohaskell/NeoHaskell#readme).
