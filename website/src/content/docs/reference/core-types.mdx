This page is a direct reference for the foundational types in NeoHaskell's standard library. Most of these are available unqualified via `import Core`.

## Text

Strict UTF-8 text.

```haskell
import Text qualified
```

| Function | Signature | Description |
|---|---|---|
| `Text.fromLinkedList` | `[Char] -> Text` | Convert a legacy `String` to `Text`. |
| `Text.toLinkedList` | `Text -> [Char]` | Convert a `Text` to a legacy `String`. |
| `Text.length` | `Text -> Int` | Character count. |
| `Text.isEmpty` | `Text -> Bool` | `True` if zero characters. |
| `Text.trim` | `Text -> Text` | Strip leading and trailing whitespace. |
| `Text.toUpper` | `Text -> Text` | Uppercase. |
| `Text.toLower` | `Text -> Text` | Lowercase. |
| `Text.contains` | `Text -> Text -> Bool` | Substring test. |
| `Text.startsWith` | `Text -> Text -> Bool` | Prefix test. |
| `Text.endsWith` | `Text -> Text -> Bool` | Suffix test. |
| `Text.split` | `Text -> Text -> Array Text` | Split by delimiter. |
| `Text.join` | `Text -> Array Text -> Text` | Join with separator. |
| `Text.replace` | `Text -> Text -> Text -> Text` | Find and replace. |
| `Text.toBytes` | `Text -> Bytes` | Encode as UTF-8. |

### String interpolation

```haskell
name = "Alice"
age = 30
message = [fmt|#{name} is #{age} years old|]
```

Use `[fmt|...|]` for any string with interpolation. Never use `++`.


| Function | Signature | Description |
|---|---|---|

| `Result.mapError` | `(err -> new) -> Result err val -> Result new val` | Transform the error. |
| `Result.andThen` | `(val -> Result err new) -> Result err val -> Result err new` | Chain. |

## Task

```haskell
newtype Task err value = Task { runTask :: ExceptT err IO value }
```

Asynchronous computation with a typed error channel.

```haskell
import Task qualified
```

| Function | Signature | Description |
|---|---|---|
| `Task.yield` | `value -> Task err value` | Return a value. |
| `Task.throw` | `err -> Task err value` | Fail with an error. |
| `Task.map` | `(input -> output) -> Task err input -> Task err output` | Transform the success value. |
| `Task.mapError` | `(err -> newErr) -> Task err value -> Task newErr value` | Transform the error. |
| `Task.andThen` | `(value -> Task err newVal) -> Task err value -> Task err newVal` | Chain (monadic bind). |
| `Task.recover` | `(err -> Task err value) -> Task err value -> Task err value` | Run a recovery function on failure. |
| `Task.asResult` | `Task err value -> Task never (Result err value)` | Convert failure into a `Result`. |
| `Task.finally` | `Task err' Unit -> Task err value -> Task err value` | Run cleanup regardless of outcome. |
| `Task.forEach` | `Array a -> (a -> Task err Unit) -> Task err Unit` | Iterate. |
| `Task.mapArray` | `(a -> Task err b) -> Array a -> Task err (Array b)` | Sequential map. |
| `Task.forever` | `Task err Unit -> Task err Unit` | Run forever (until error). |
| `Task.fromIO` | `IO a -> Task err a` | Wrap a raw `IO` action. |
| `Task.fromFailableIO` | `IO a -> Task Text a` | Wrap an `IO` action that might throw. |
