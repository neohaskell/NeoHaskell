## What is NeoHaskell?

NeoHaskell is a dialect of Haskell designed for beginners and pragmatists. It makes
event sourcing a first-class language feature: events, entities, commands, queries,
services, and integrations are language primitives, not library patterns. If a state
transition is wrong, the program does not compile.

You write the **core** — the business rules that make your product yours. NeoHaskell
handles the **shell** around them: HTTP endpoints, JSON schemas, CLI parsers, event
persistence, and a complete audit trail.
