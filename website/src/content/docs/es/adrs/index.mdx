Los Registros de Decisiones Arquitectónicas (ADR) documentan las decisiones arquitectónicas significativas tomadas al construir NeoHaskell.

Los registros ADR se mantienen de forma canónica **en inglés** y no se traducen, para conservar una única fuente de verdad. Consulta la lista completa, con una página por cada decisión:

- **[Registros de Decisiones Arquitectónicas (en inglés)](/adrs/)**
