NeoHaskell vive en el monorepo [neohaskell/NeoHaskell](https://github.com/neohaskell/NeoHaskell). Este sitio de documentación es intencionadamente pequeño; las instrucciones autoritativas de instalación, compilación y pruebas están en el `README.md` del repositorio (en inglés).

## Inicio rápido

1. Instala [Determinate Nix](https://determinate.systems/nix).
2. Clona el repositorio y entra en su shell de desarrollo con `nix develop`.
3. Ejecuta `./dev` para ver los comandos de compilación y prueba disponibles.

Para el flujo de trabajo completo —Postgres, la suite de pruebas y el proceso de contribución— consulta el [README del repositorio](https://github.com/neohaskell/NeoHaskell#readme).
